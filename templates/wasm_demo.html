About This Demo

This page demonstrates how Rust code can be compiled to WebAssembly and run directly in your browser. The functions above are all written in Rust and compiled to WebAssembly, providing:

  • Near-native performance
  • Strong type safety
  • Memory safety without garbage collection
  • Smaller payload sizes compared to JavaScript libraries

The source code for these functions is available in the wasm directory of this project.
